MS Playground

About the Playground

MS Playground is a web service like the Go Playground. Similarly, it receives a Go program, compiles, links, and runs the program inside a sandbox, then returns the output.

The source code is published on Github, and examples can be found on the MS Playground wiki.

MS Playground targets mass spectrometry programs. It differs from the Go Playground in three ways.

  • It can also serve executables of the compiled programs. The download button compiles six binaries, for Windows, Mac OS X, and Linux, on 32-bit and 64-bit architectures. It are command-line tools that accept flags.
  • There are two test files available on the server. They can be loaded with the unthermo library. The files are called "small.RAW" and "itraq.RAW". Any attempt to load another file will fail.
  • The playground can display a 2D scatter plot instead of raw numbers. The plot script expects two columns of numbers, separated by space.

Like on the Go Playground, there are limitations to the programs that can be run. These limitations don't apply to the served executables.
